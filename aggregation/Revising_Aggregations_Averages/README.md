## Revising Aggregations - Averages

#### Difficulty: Easy

</br>Query the average population of all cities in **CITY** where District is **California**.

<br>**Input Format**<br>
<br>The **CITY** table is described as follows:
<br><br>

**Table: CITY**

| Field       | Type         |
| ----------- | ------------ |
| ID          | NUMBER       |
| NAME        | VARCHAR2(17) |
| COUNTRYCODE | VARCHAR2(3)  |
| DISTRICT    | VARCHAR2(20) |
| POPULATION  | NUMBER       |
