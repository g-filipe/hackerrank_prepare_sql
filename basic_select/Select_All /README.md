## Select All

#### Difficulty: Easy

</br>Query all columns (attributes) for every row in the **CITY** table.

The CITY table is described as follows:
<br><br>

**Table: CITY**

| Field       | Type         |
| ----------- | ------------ |
| ID          | NUMBER       |
| NAME        | VARCHAR2(17) |
| COUNTRYCODE | VARCHAR2(3)  |
| DISTRICT    | VARCHAR2(20) |
| POPULATION  | NUMBER       |
